# Development Standards

Please ensure the following standards are met for all codebase changes:

1.  **MkDocs**: Update the documentation in `docs/` and `mkdocs.yml` to reflect code changes.
2.  **Sample Notebook**: Add or update a Jupyter notebook (e.g., in `notebooks/`) to demonstrate usage.
3.  **Pytest**: Add or update tests using `pytest` to ensure code correctness.
