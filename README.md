# ig-api
API for trading

## Setup using uv

This project uses `uv` for dependency management.

1.  **Install uv** (if not installed):
    ```bash
    brew install uv
    ```

2.  **Sync dependencies**:
    ```bash
    uv sync
    ```

3.  **Run the script**:
    ```bash
    uv run main.py
    ```
